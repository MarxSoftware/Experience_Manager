== Description ==

This Plugin integrates the ecommerce-engine into your WordPress/WooCommerce installation.

Features:

* Different kinds of product recommendations
* Display recommendations on product detail page
* Display recommendations on shop page
* Display recommendations on category page


== Installation ==

This section describes how to install the plugin and get it working.
You need an active account for our ecommerce-engine https://marx-software/ecommerce/ecommerce-engine

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "Experience Manager" menu to configure the plugin

== Usage ==
The Experience Manager allows you to define different kinds of product recommendations on different postions in your WooCommerce shop.




== Frequently Asked Questions ==

= Can I use the plugin without ecommerce-engine? =
No, this is no possible. The ecommerce-engine is needed to generate product recommendations.

= Where can I get an account for the ecommerce-engine? =
Visit https://marx-software.de/ecommerce/ecommerce-engine

= Is this addon compatible with other WordPress cache addons like WP Super Cache? =
Yes, Experience Manager is compatible to all cache plugins.
